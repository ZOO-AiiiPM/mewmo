# Interaction Spec

累积记录 mewmo app 的交互规范。每条精确描述触发条件 → 期望行为。

## Checkbox Toolbar - Toggle 行为

- **触发条件**：点击 toolbar 的勾选框按钮
- **已是任务项时（`- [ ]` 或 `- [x]`）**：移除整个 checkbox 前缀，变回纯文本
- **纯文本/无序列表时**：加上 `- [ ]` 前缀
- **本质是 toggle**：在「纯文本 ↔ 任务项」之间切换，不是在「选中 ↔ 未选中」之间切换
- **多行同理**：选中多行时对每行独立 toggle

## 侧边栏 Zone Tab - 新增页面

- **触发条件**：点击侧边栏的 zone tab（订阅/笔记/剪藏/沉淀）
- **已有该 zone 的 tab 存在时**：直接切换到该 tab（不重复创建）
- **没有该 zone 的 tab 时**：在主区域新增一个 tab 页并激活
- **笔记 zone 特殊处理**：新增 tab 时自动打开最新一条笔记

## Tab 栏关闭按钮

- **选中 tab**：叉号始终显示
- **未选中 tab**：叉号隐藏且不占宽度
- **hover 未选中 tab 时**：叉号出现
